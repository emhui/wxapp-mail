# wxapp-mail

#### 介绍

此项目参考 [https://github.com/lin-xin/wxapp-mall/tree/uni](https://github.com/lin-xin/wxapp-mall/tree/uni)
使用微信小程序模范了一个购物APP。目前有下面几个功能

- 新品列表
- 商品详情
- 商品分类
- 商品搜索
- 购物车
- 个人信息
- ...
